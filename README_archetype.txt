This project was created using the Vaadin add-on archetype.

The current version of this archetype supports server side add-ons only.

After creating a project with the archetype, perform the following steps to create and publish an add-on.

- Update project metadata in parent/pom.xml (organization, any additional information)
- Write add-on code in addon/src/main/java
- Write add-on demo application code in demo/src/main/java
- Write the manual for the add-on in manual/src/docbkx/manual.xml
- Update licensing.txt and README.txt
- Follow the instructions in README.txt to build and publish the add-on
